Battleship Game
A classic Battleship game for Android, featuring multiplayer, animations, and a modern UI. Built natively for a smooth mobile experience.
Project Details / Background
The Battleship Game is a modern take on the classic naval warfare game, developed natively for Android. The game features a sleek, intuitive interface with smooth animations and engaging sound effects. Players can enjoy both local multiplayer and AI opponents, with different difficulty levels to challenge their strategic thinking.
My role in this project involved designing the game architecture, implementing the core game logic, and creating an engaging user interface. The project showcases my ability to develop polished mobile applications with attention to user experience and performance optimization.
