EnergySales WebApp
A full-stack web application for energy sales management, built with Ktor (Kotlin) backend and React frontend. Features dashboards, analytics, and user management.
Project Details / Background
EnergySales WebApp was designed to streamline the process of managing energy sales, contracts, and analytics for energy companies. The backend is powered by Ktor (Kotlin), providing a robust API, while the frontend leverages React for a dynamic and responsive user experience. The app includes real-time dashboards, user authentication, and detailed reporting tools.
My role included designing the system architecture, implementing RESTful APIs, and building interactive UI components. The project demonstrates my ability to deliver scalable, maintainable, and user-friendly web solutions.
